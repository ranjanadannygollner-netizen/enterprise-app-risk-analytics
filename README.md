# enterprise-app-risk-analytics
Enterprise Application Compatibility &amp; Risk Analysis using Excel and SQL
**Project Overview**

This project analyzes 1200+ enterprise applications to evaluate compatibility risks, version gaps, and upgrade status.

**Tools Used**

Excel (Pivot Tables, KPI Metrics, Risk Modeling)

SQL (Data Querying & Aggregation)

**Key Insights**

Identified high-risk business units

Analyzed upgrade backlog distribution

Developed KPI-driven dashboard for stakeholder reporting
